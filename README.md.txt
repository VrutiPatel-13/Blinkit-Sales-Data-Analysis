# 🛒 Blinkit Sales Data Analysis Dashboard

## 📊 Project Overview
Developed a comprehensive data analytics solution to analyze Blinkit (grocery delivery) sales data and generate actionable business insights. The project focuses on understanding sales trends, product demand, and outlet performance to support data-driven decision-making.

---

## 🎯 Objectives
- Analyze overall sales performance and revenue trends  
- Identify high-demand products and categories  
- Evaluate outlet-level performance  
- Understand seasonal and category-wise sales patterns  

---

## 🛠️ Tools & Technologies Used
- **Excel** – Data preprocessing and initial exploration  
- **SQL** – Data extraction, transformation, and querying  
- **Python (Pandas, Matplotlib)** – Data cleaning and exploratory data analysis (EDA)  
- **Power BI** – Interactive dashboard creation and visualization  

---

## 🔍 Key Analysis Performed
- Sales trend analysis (monthly/yearly)  
- Product and category-level performance  
- Outlet-wise sales and profitability analysis  
- Seasonal demand pattern identification  
- Data cleaning and transformation for accurate insights  

---

## 📊 Dashboard Features
- KPI Cards: Total Sales, Profit, Orders  
- Sales Trend (Line Chart)  
- Category-wise Sales (Donut/Bar Chart)  
- Outlet Performance Analysis  
- Product Demand Insights  
- Interactive filters (Date, Category, Outlet)
